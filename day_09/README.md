# Day 9 - Functions, Pointers and Assessment Tests

## Topics Covered

### 1. Functions in C

Learned how functions help in breaking a program into smaller, reusable modules.

Concepts covered:

* Function declaration
* Function definition
* Function calling
* Parameters and arguments
* Return values

Benefits:

* Code reusability
* Better readability
* Easier debugging and maintenance

---

### 2. HCF (Highest Common Factor)

Implemented a program to find the HCF of two numbers using a function.

Concepts used:

* User-defined functions
* Euclidean Algorithm
* Loops
* Return statements

Example:

```text
Input : 24 36
Output: HCF = 12
```

---

### 3. Pointers

Introduced to pointers and memory address manipulation.

Concepts covered:

* Pointer declaration
* Address operator (&)
* Dereference operator (*)
* Passing addresses to functions

Example:

```c
int *ptr;
```

---

### 4. Call by Reference

Learned how to modify original variables by passing their addresses to functions.

Concepts used:

* Pointers as function parameters
* Memory referencing
* Data manipulation through addresses

---

### 5. Swapping Two Numbers Using Pointers

Implemented a program to swap two numbers using call by reference.

Concepts used:

* Pointers
* Functions
* Temporary variables
* Address passing

Example:

```text
Before: 10 20
After : 20 10
```
