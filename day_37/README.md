# Day 37 - Binary Search Tree Deletion & LeetCode Tree Problems

## 📅 Overview

Day 37 focused on implementing the **Deletion operation in a Binary Search Tree (BST)** and strengthening tree concepts by solving **Binary Tree and BST problems on LeetCode**. The session emphasized understanding the different deletion cases in BSTs and applying tree traversal and recursion techniques to coding problems.

---

## 📚 Topics Covered

### 1. Binary Search Tree (BST) Deletion

Studied and implemented deletion in a Binary Search Tree.

#### Deletion Cases

##### Case 1: Leaf Node

- Node has no children.
- Delete the node directly.

##### Case 2: Node with One Child

- Replace the node with its child.
- Delete the original node.

##### Case 3: Node with Two Children

- Find the **Inorder Successor** (minimum value in the right subtree) or **Inorder Predecessor** (maximum value in the left subtree).
- Replace the node's value.
- Delete the successor/predecessor node recursively.
