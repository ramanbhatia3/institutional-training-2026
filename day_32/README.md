# Day 32 - Queue Variants & Priority Queue

## 📅 Overview

Day 32 focused on advanced queue data structures and their implementations using both **arrays** and **linked lists**. The concepts covered included **Queue using Linked List**, **Circular Queue using Linked List**, **Double Ended Queue (Deque)**, and **Priority Queue**.

---

## 📚 Topics Covered

### 1. Queue using Linked List

Implemented a queue using a singly linked list.

#### Operations Implemented

- Enqueue
- Dequeue
- Peek
- Display
- Check if Queue is Empty

---

### 2. Circular Queue using Linked List

Implemented a Circular Queue where the last node points back to the first node.

#### Operations Implemented

- Enqueue
- Dequeue
- Peek
- Display
- Check if Queue is Empty

---

### 3. Double Ended Queue (Deque)

Implemented a basic Deque using arrays.

#### Operations Implemented

- Enqueue at Front
- Enqueue at Rear
- Dequeue from Front
- Dequeue from Rear
- Display
- Check if Queue is Empty
- Check if Queue is Full

---

### 4. Priority Queue

Implemented Priority Queue using two approaches.

#### Using Arrays

- Enqueue
- Dequeue (Highest Priority Element)
- Peek
- Display
- Check if Queue is Empty
- Check if Queue is Full

#### Using Linked List

- Sorted Insertion
- Dequeue (Highest Priority Element)
- Display
- Check if Queue is Empty
