# Day 6 - Mathematical Programs, Prime Numbers and Revision

## Topics Covered

### 1. Quadratic Equation Roots

Implemented a program to find the roots of a quadratic equation:

```text
ax² + bx + c = 0
```

Concepts used:

* Discriminant calculation
* Conditional statements
* Mathematical functions (`sqrt()`)
* Real and imaginary roots

Cases handled:

* Two distinct real roots
* One repeated real root
* Two complex roots

Formula Used:

```text
D = b² - 4ac
```

```text
Root = (-b ± √D) / 2a
```
