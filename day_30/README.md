# Day 30 - STL Containers (Array, List & Vector)

## 📅 Overview

Day 30 introduced the **Standard Template Library (STL)** in C++. The focus was on understanding three of the most commonly used STL containers:

- `std::array`
- `std::list`
- `std::vector`

Through hands-on coding, various built-in member functions, iterators, and algorithms were explored to understand how STL simplifies programming and improves code efficiency.
