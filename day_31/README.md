# Day 31 - Queue Implementation (Linear & Circular Queue)

## 📅 Overview

Day 31 focused on understanding the **Queue** data structure and its different implementations. Programs were developed in both **C** and **C++** to implement **Linear Queue** and **Circular Queue**, covering all fundamental queue operations.

---

## 📚 Topics Covered

### 1. Linear Queue

Implemented using arrays in C++.

#### Operations Implemented

- Enqueue (Insert)
- Dequeue (Delete)
- Display Queue
- Check if Queue is Empty
- Check if Queue is Full

---

### 2. Circular Queue

Implemented in both **C** and **C++** using arrays.

#### Operations Implemented

- Enqueue
- Dequeue
- Peek (Front Element)
- Display Queue
- Check if Queue is Empty
- Check if Queue is Full

The circular queue implementation uses modulo (`%`) arithmetic to efficiently utilize memory by reusing empty spaces created after deletions.

---

## Concepts Learned

- FIFO (First In First Out) Principle
- Linear Queue
- Circular Queue
- Front and Rear Pointers
- Queue Overflow
- Queue Underflow
- Circular Traversal using Modulo Operator
- Dynamic Memory Allocation using Arrays
- Menu Driven Programming
