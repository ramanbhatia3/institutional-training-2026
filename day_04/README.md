# Day 4 - Conditional Statements and Switch Case in C

## Topics Covered

### Decision Making in C

Decision-making statements allow a program to execute different blocks of code based on specified conditions.

---

## if Statement

The `if` statement executes a block of code only when a condition is true.

Example:

```c
if(condition){
    // code
}
```

Used for:

* Checking positive or negative numbers
* Comparing values
* Validating conditions

---

## if-else Statement

The `if-else` statement provides two execution paths depending on whether the condition is true or false.

Example:

```c
if(condition){
    // true block
}
else{
    // false block
}
```

---

## else-if Ladder

Used when multiple conditions need to be checked.

Example:

```c
if(condition1){
}
else if(condition2){
}
else{
}
```

Applications:

* Number classification
* Grade calculation
* Menu-driven programs

---

## Finding Maximum Value

Conditional statements can be used to compare multiple numbers and determine the largest value.

Concepts used:

* Comparison Operators
* Decision Making
* Variable Updating

---

## Switch Statement

The `switch` statement is used as an alternative to multiple `if-else` statements when comparing a single variable against multiple values.

Syntax:

```c
switch(expression){
    case value1:
        break;

    case value2:
        break;

    default:
        break;
}
```

---

## Break Statement

The `break` statement terminates the current case and prevents execution from continuing into the next case.

---

## Default Case

Executed when no matching case is found.

Example:

```c
default:
    printf("Invalid Input");
```
