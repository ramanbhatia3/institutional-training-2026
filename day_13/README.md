# Day 13 - Structures in C and Self-Referential Structures

## Topics Covered

### 1. Structures in C

Learned how structures allow us to group multiple variables of different data types under a single user-defined data type.

Concepts Covered:

- Structure declaration
- Structure initialization
- Accessing members using the dot (`.`) operator

---

### 2. Array of Structures

Implemented programs to store and manage multiple employee records using arrays of structures.

Topics covered:

- Multiple records
- User input
- Displaying structure data
- Iterating through structure arrays

---

### 3. Structure Padding

Learned how the compiler inserts extra memory (padding) for alignment.

Topics covered:

- Memory alignment
- `sizeof()` operator
- `#pragma pack(1)` to remove padding

---

### 4. Structure Pointers

Practiced accessing structure members using pointers.

Topics covered:

- Pointer to structure
- Arrow (`->`) operator
- Passing structure addresses

---

### 5. Input using Structure Pointer

Learned to take user input directly through structure pointers.

Concepts covered:

- Dynamic member access
- Pointer notation
- Cleaner input handling

---

### 6. Self-Referential Structures

Studied structures containing pointers to the same structure type.

Topics covered:

- Self-referential structures
- Linking structure objects
- Sequential traversal using pointers
- Introduction to the concept behind linked lists

---
