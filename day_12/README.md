# Day 12 - Linked List Practice for DSA

## Topics Covered

### 1. Introduction to Linked Lists

- Understanding the structure of a singly linked list.
- Creating nodes dynamically using `malloc()`.
- Linking multiple nodes together.
- Traversing and displaying a linked list.

---

### 2. Dynamic Linked List Creation

Implemented a program to:

- Create a linked list based on user input.
- Insert nodes dynamically.
- Traverse and print all elements.

---

### 3. Reversing a Linked List

Learned the iterative approach to reverse a singly linked list using three pointers:

- `prev`
- `current`
- `next`

Concepts covered:

- Pointer manipulation
- In-place reversal
- Efficient traversal

---

### 4. Addition of Two Numbers using Linked Lists

Implemented a program to add two numbers represented as linked lists.

Topics covered:

- Node creation
- Carry handling
- Building a new result list
- Traversing multiple linked lists simultaneously
