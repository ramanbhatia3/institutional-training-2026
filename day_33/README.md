# Day 33 - Recursion & Introduction to Analysis of Algorithms

## 📅 Overview

Day 33 marked the beginning of **Analysis of Algorithms (AOA)** along with practicing various **Recursion** problems in C++. The session introduced algorithm efficiency through **Time Complexity** and **Space Complexity**, followed by the study of **Asymptotic Notations**. Several recursion examples were implemented to understand recursive function calls, recursion trees, and base cases.

---

## 📚 Topics Covered

### 1. Recursion

Practiced different recursion-based programs to understand recursive function execution.

#### Programs Implemented

- Basic function calling
- Nested function calls
- Recursive printing (ascending & descending order)
- Recursive multiplication
- Mutual Recursion
- Recursive doubling example
- Fibonacci Sequence using Recursion
- Decimal to Binary Conversion using Recursion

---

### 2. Analysis of Algorithms (AOA)

Introduction to algorithm analysis and efficiency.

#### Topics Studied

- What is an Algorithm?
- Efficiency of an Algorithm
- Time Complexity
- Space Complexity

---

### 3. Asymptotic Notations

Studied the three major asymptotic notations used to analyze algorithm growth.

- **Big O (O)** – Upper Bound (Worst Case)
- **Big Omega (Ω)** – Lower Bound (Best Case)
- **Big Theta (Θ)** – Tight Bound (Average/Exact Growth)

---

### 4. Growth of Functions

Compared the growth rates of different mathematical functions.

Examples discussed:

- \(f(n)=n+4\) vs \(g(n)=n^2\)
- \(f(n)=2^n\) vs \(g(n)=n^2\)
- Logarithmic growth functions

Learned that **constant factors do not affect asymptotic growth**, while the order of growth determines algorithm efficiency for large input sizes.
